**leagueinfo.gg** é uma aplicação web de alta performance construída com React e Vite, focada em fornecer informações detalhadas sobre os campeões de League of Legends. O projeto prioriza a experiência do usuário através de métricas rigorosas de performance, acessibilidade e SEO

## ✨ Funcionalidades

* **Exploração de Campeões:** Listagem completa consumindo a API oficial (Data Dragon) com filtros dinâmicos por classe.
* **Informações Detalhadas:** Visualização de habilidades, skins, história e dicas de aliados/inimigos.
* **Suporte Multi-idioma:** Disponívem em português e inglês
* **Design Responsivo:** Interface moderna e adaptável para qualquer tamanho de tela.

## Tecnologias Utilizadas

* [React](https://reactjs.org/) (Vite)
* [React Router](https://reactrouter.com/) (Roteamento SPA)
* CSS 3
* [React Helmet Async](https://github.com/staylor/react-helmet-async) (para SEO Dinâmico)
* [Data Dragon API](https://developer.riotgames.com/docs/lol) (Riot Games)
* [Weserv.nl](https://images.weserv.nl/) (Image Optimization Proxy)

## 🔧 Como rodar o projeto

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/SEU_USUARIO/leagueinfo.gg.git](https://github.com/SEU_USUARIO/leagueinfo.gg.git)
    ```
2.  **Entre na pasta:**
    ```bash
    cd leagueinfo.gg
    ```
3.  **Instale as dependências:**
    ```bash
    npm install
    ```
4.  **Inicie o ambiente de desenvolvimento:**
    ```bash
    npm run dev
    ```
5.  **Para testar a versão de produção:**
    ```bash
    npm run build
    npm run preview
    ```
